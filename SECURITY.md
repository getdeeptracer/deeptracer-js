# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.4.x   | Yes       |
| < 0.4   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in the DeepTracer SDK, please report it responsibly.

**Do NOT open a public GitHub issue.**

Instead, email **security@deeptracer.dev** with:

- A description of the vulnerability
- Steps to reproduce
- Affected packages and versions
- Any potential impact

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

## Scope

This policy covers all `@deeptracer/*` npm packages in this repository:

- `@deeptracer/core`
- `@deeptracer/node`
- `@deeptracer/ai`
- `@deeptracer/browser`
- `@deeptracer/react`
- `@deeptracer/nextjs`
